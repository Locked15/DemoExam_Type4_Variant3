# ООО «Руль»

ООО «Руль» - магазин по продаже автомобильных товаров.
В рамках выполнения демонстрационного экзамена был реализован следующий функционал:

* неавторизованный клиент и авторизованный клиент может просматривать товары и
формировать заказ;
* менеджер может просматривать товары, формировать и редактировать заказы;
* администратор может добавлять/редактировать/удалять товары, просматривать и
редактировать заказы.

## Руководство пользователя

Для начала работы с приложением необходимо запустить файл "SteeringWheelApp.exe".

## Использованные технологии

* Язык программирования — C#;
* СУБД — MySQL;
* IDE — Visual Studio 2022.
